- Open the CAPP App folder in the location where you saved it.
Launch Command Prompt (cmd).

- Navigate to the drive where the app folder is located (e.g., type cd desktop).

- Access the app folder (e.g., type cd cattle_antibody_prediction_updated_version).

- Run the Python script for the app by entering: streamlit run cattle_antibody_prediction_with_combinatory_analysis.py.

- Once the commands are executed, a browser window will open with the app dashboard.

- Upload the antibody sequence dataset in CSV format to run predictions.
